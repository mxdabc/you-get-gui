# You-get 的 Gui 版本

## 主要特性
- 支持下载各大视频平台视频
- 支持Windows 11
- 去除原版 you-get 的命令行下载的繁琐
- 易用、简单、易上手

## 安装与使用
- 点击仓库旁的 Release 或点击→ [Release](https://github.com/mxd0123456/you-get-gui/releases "Release")
- 找到以Setup_XXXX开头的软件包
- 点击下载
- 也可使用绿色版，单文件，免解压 文件名：Green_XXXX
